 🏬 Mall Navigation System

 📌 Overview
The Mall Navigation System is a mobile application designed to help users easily navigate large shopping malls. It provides an interactive map and guides users to their desired stores, services, or facilities within the mall.

---

 🚀 Features
- 📍 Search for stores and services
- 🗺️ Interactive indoor navigation
- 🧭 Shortest path guidance
- 🏢 Multi-floor navigation support
- 🔎 Category-based browsing (food, fashion, electronics, etc.)
- 🚻 Locate facilities like washrooms, ATMs, parking

---

 🛠️ Tech Stack
- **Frontend:** React Native (Expo)
- **Backend:** Node.js (if applicable)
- **Navigation:** Custom routing / map-based logic
- **Tools:** JavaScript, VS Code
